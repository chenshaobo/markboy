Markdown 仅负责生成文档的结构，而不处理样式。就如 HTML 和 CSS 的关系一样――实际上，Markdown 就可以看作是简化的 HTML, 其诞生的目的，就是为了便于在写作时对文档进行格式化，并且使得用 Markdown 写成的纯文本要比 HTML 可读。
